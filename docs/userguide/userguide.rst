User's Guide
========================

This guide provides documentation of how to use One and Done. We will be adding
documents which will explain some best practices for users as well as developers
interested in setting up One and Done locally.

If you find an issue with One and Done, please enter a bug in `Bugzilla`_

.. _Bugzilla: https://bugzilla.mozilla.org/enter_bug.cgi?component=OneandDone&product=Mozilla%20QA
