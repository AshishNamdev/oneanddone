
Welcome to One and Done
=======================

One and Done is a tool built by Mozilla.  One and Done matches new contributors with tasks to help out a variety of Mozilla teams.

Useful links
------------

* You can browse around on our `staging`_ site
* Or download it from `github`_ and run it locally
* Feel free to ask questions: irc.mozilla.org #mozwebqa
* If you find any bugs, please enter them in `Bugzilla`_

.. _staging: https://oneanddone.allizom.org/
.. _github: https://github.com/mozilla/oneanddone/
.. _Bugzilla: https://bugzilla.mozilla.org/enter_bug.cgi?&component=One%20and%20Done&product=Mozilla%20QA
